# zpm-ui

> Zpm-Ui组件库

[文档](http://zpmui.dei2.com)

## Usage

``` bash
# install dependencies
npm install zpm-ui

# 导入zpm-ui
import ZpmUi from 'zpm-ui'

# 使用组件
Vue.use(ZpmUi)
```
